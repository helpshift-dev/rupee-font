Rupee Font


Use the Rupee symbol on your website the right way

While there are many approaches to using the Rupee symbol on a web page, we have come up with a different way to do it in a backwards compatible yet unobtrusive manner.

We created a custom font with only one code-point (U+20A8) which stands for RUPEE SIGN and we assigned the nice Rupee symbol glyph designed by Foradian Technologies to that code-point. Using that font is easy, just use &#8360; wherever you need the Rupee symbol and wrap it in a span tag with a CSS class; finally through CSS one can use the custom font to style the content. If a browser doesn’t support @font-face the user will see the ₨ symbol which is widely supported. This is one of the reasons why we used that code-point instead of the official INDIAN RUPEE SIGN code-point U+20B9 (another reason being the fact that our font simply predated the assignment of the new code-point).

The advantage of this approach is that it's easy to use and fully backwards compatible. You can see it in action on Paisa.com — a webapp about the Indian stock market.

Using the font on your website is easy, just include our stylesheet somewhere in your HTML page like this:

<link rel="stylesheet" type="text/css" media="screen" href="http://qotd.github.com/rupee-font/rupee.css" />

Then you can use the following markup wherever you need the Rupee symbol:

<span class="rupee">&#8360<span>

And the text should show up like this: ₨

You can also fork the project on Github.
